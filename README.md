\# Mast Automation



Encoder-driven stepper motor mast / quill controller.



\## Features

\- Manual encoder stepping (1 step per detent)

\- OLED height display

\- Save / Load positions (EEPROM)

\- Go-to moves with cancel

\- Zero calibration

\- Position persistence across restart



\## Hardware

\- Arduino Uno

\- DRV8825

\- NEMA 17 stepper

\- SSD1306 128x32 OLED

\- Mechanical rotary encoder + button



\## Status

Working baseline. Refactor planned.
